# Microsoft Azure - Localizando Serviços por Categoria

## Resumo do Lab

Neste laboratório aprendi os conceitos básicos da plataforma Microsoft Azure e como localizar os serviços disponíveis por categoria dentro do portal Azure.

Durante o desenvolvimento do lab, entendi que o Azure é uma plataforma de computação em nuvem da Microsoft que oferece diversos serviços para desenvolvimento, armazenamento, redes, banco de dados, inteligência artificial e muito mais.

Também aprendi a navegar pelo portal do Azure, identificando os serviços organizados por categorias, o que facilita encontrar recursos específicos para cada necessidade de negócio ou desenvolvimento.

### Principais aprendizados

* Como acessar e utilizar o portal Microsoft Azure;
* Entendimento sobre computação em nuvem;
* Organização dos serviços do Azure por categorias;
* Diferença entre serviços de:

  * Computação;
  * Rede;
  * Armazenamento;
  * Banco de Dados;
  * Segurança;
  * Inteligência Artificial;
* Importância da nuvem para escalabilidade e disponibilidade de sistemas;
* Como localizar rapidamente recursos dentro da plataforma Azure.

### Conclusão

Esse laboratório foi importante para compreender melhor a estrutura da plataforma Microsoft Azure e como os serviços são organizados. O conhecimento adquirido será útil para futuros projetos utilizando computação em nuvem e serviços Microsoft.
